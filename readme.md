## 🚀 Demonstração Visual e Funcional

O projeto conta com uma interface imersiva inspirada na temática espacial e estética cyberpunk, utilizando um plano de fundo em vídeo dinâmico, efeitos visuais refinados de vidro soprado (Glassmorphism) e feedback sonoro premium nas interações.

> 🔊 **Diferencial de UX:** Navegação fluida com transição em *Fade In/Fade Out* sincronizada com efeitos sonoros nativos (`.mp3`) de clique via JavaScript assíncrono.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

### Front-End
*   **HTML5 & CSS3:** Estruturação semântica e estilização customizada com arquitetura modular de arquivos (`base.css`, `layout.css`, `componentes.css`).
*   **JavaScript (ES6+):** Manipulação dinâmica do DOM, gerenciamento de estados do menu mobile (hambúrguer) e manipulação da API de Áudio nativa para feedback sonoro sem latência.
*   **Google Fonts & Font Awesome:** Integração tipográfica com a fonte *Inter* e pacote de ícones vetoriais de alta fidelidade para as redes sociais.

### Back-End & Conhecimentos Integrados (Foco da Carreira)
*   **Python, Flask & Django:** Estruturação de APIs robustas e gerenciamento lógico por trás dos panos.
*   **MySQL:** Modelagem e integração inteligente com bancos de dados relacionais.
*   **Git & GitHub:** Versionamento rígido de código e controle de ramificações (branches).
