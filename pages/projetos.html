◉Projeto Recente: To-Do List Cloud

Uma aplicação web moderna de gerenciamento de tarefas que evoluiu de um armazenamento local simples para uma arquitetura completa cliente-servidor com persistência de dados em nuvem.

☩Interface & Funcionalidades

Desenvolvimento focado em dinamismo, usabilidade e sincronização em tempo real:

  • ●Gerenciamento Ágil: Adição, conclusão e remoção de tarefas com atualizações visuais instantâneas.
  • ●Sincronização Multiplataforma: Acesso responsivo de qualquer dispositivo com dados atualizados em tempo real.
  • ●API de Motivação Integrada: Consumo de API internacional de conselhos via servidor Proxy com tradução dinâmica no backend.

☩Arquitetura & Cloud

Estrutura resiliente projetada para garantir a segurança e a integridade dos dados:

  • ●Persistência na Nuvem: Integração com banco de dados relacional online, evitando perda de dados ao limpar o navegador.
  • ●Histórico Automatizado: Banco de dados estruturado com gatilhos que salvam tarefas removidas em uma tabela de auditoria com data e horário.
  • ●Evolução de Stack: Migração de arquitetura monolítica local (LocalStorage) para o modelo distribuído Cliente-Servidor.
